◆ auto_calibracion()

void auto_calibracion ( void )

Realiza auto-calibración inicial de sensores.

Calibra los valores mínimos y máximos de los sensores IR

Realiza auto-calibración inicial de sensores.

Proceso de calibración en 3 etapas con indicación led:

Etapa 1 (LED Naranja): Cerca de pared derecha (3s)

  • Captura valor mínimo del sensor derecho (der_cerca)

Etapa 2 (LED Rojo): Cerca de pared izquierda (3s)

  • Captura valor mínimo del sensor izquierdo (izq_cerca)

Etapa 3 (LED Azul): Centrado en pasillo (3s)

  • Captura valores máximos de ambos sensores (izq_lejos, der_lejos)
  • Calcula puntos medios para detección de centrado

Finalización (LED Verde): Calibración completa

  • Activa flag calibrado = true
Atención
Requiere intervención manual para posicionar el robot
Nota
Los umbrales se calculan dinámicamente según las condiciones del entorno

void chequeolinea ( void )

Procesa la detección de una línea.

Procesa la detección de una línea del laberinto.

Actualiza posición, verifica meta, calcula nueva dirección y ejecuta movimiento

Secuencia completa de procesamiento al cruzar una línea:

  1. Desactiva interrupciones temporalmente
  2. Avanza el tiempo necesario para posicionarse
  3. Actualiza la posición del robot
  4. Verifica si llegó a la meta (1,1)
  5. Calcula la mejor dirección usando Flood Fill
  6. Ejecuta el movimiento necesario
  7. Reactiva interrupciones
Nota
Usa TIEMPO_AVANCE_LINEA que varía según el modo (exploración/sprint)

◆ chequeomuro()

void chequeomuro ( void )

Procesa la detección de un muro.

Registra el muro, recalcula pesos y ejecuta nuevo movimiento

Secuencia de procesamiento al detectar un obstáculo:

  1. Desactiva interrupciones
  2. Registra el muro en el mapa del laberinto
  3. Recalcula todos los pesos usando Flood Fill
  4. Calcula nueva mejor dirección
  5. Ejecuta el movimiento alternativo
  6. Reactiva interrupciones
Nota
El algoritmo Flood Fill se ejecuta completamente tras cada muro detectado

◆ controlar_linea_recta()

void controlar_linea_recta ( void )

Controla el seguimiento de línea recta.

Mantiene el robot centrado

Controla el seguimiento de línea recta.

Algoritmo de control reactivo:

  1. Verifica que la calibración esté completa
  2. Evalúa la proximidad a las paredes laterales
  3. Aplica correcciones direccionales según la situación:
    • Muy cerca de pared izquierda → Corrección derecha
    • Muy cerca de pared derecha → Corrección izquierda
    • Centrado → Avance recto
Nota
Utiliza umbrales dinámicos calculados en calibración
Margen de seguridad de 200 unidades sobre valores de calibración
Atención
No opera sin calibración previa (calibrado = false)

void promediar_sensores ( uint16_t * buffer)

Promedia las lecturas de los sensores IR.

Parámetros
bufferBuffer con las lecturas DMA del ADC

Promedia las lecturas de los sensores IR.

Parámetros
bufferPuntero al segmento del buffer DMA a procesar

Proceso de filtrado:

  • Procesa MUESTRAS lecturas intercaladas (canal 8 y 9)
  • Canal 8 (PB0): Sensor derecho
  • Canal 9 (PB1): Sensor izquierdo
  • Calcula promedio aritmético para reducir ruido
  • Actualiza variables globales sensor_der_avg y sensor_izq_avg
Nota
Se ejecuta constantemente en DMA para actualización en tiempo real

Maneja el botón para activar modo sprint.

Reinicia posición y activa modo de alta velocidad
