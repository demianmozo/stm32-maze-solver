void laberinto_init ( void )

Inicializa el laberinto con pesos ideales y sin muros.

Configura el laberinto con pesos calculados como distancia Manhattan desde cada casilla hasta la meta (1,1). Inicialmente no hay muros.

Inicializa el laberinto con pesos ideales y sin muros.

Configura cada casilla con:

  • Identificación de posición (fila, columna)
  • Peso inicial calculado como distancia Manhattan a la meta
  • Sin muros inicialmente (laberinto abierto)
  • Meta en (1,1) con peso 0

◆ laberinto_recalcular_pesos()

void laberinto_recalcular_pesos ( void )

Recalcula todos los pesos del laberinto usando Flood Fill.

Propaga los pesos desde la meta hacia todas las casillas, respetando los muros conocidos

Recalcula todos los pesos del laberinto usando Flood Fill.

Algoritmo iterativo que propaga pesos desde la meta:

  1. Mantiene la meta con peso 0
  2. Para cada casilla, calcula peso = menor_peso_adyacente + 1
  3. Respeta muros existentes (no considera casillas bloqueadas)
  4. Itera hasta convergencia o máximo de iteraciones
Nota
Protección contra bucles infinitos con MAX_ITERACIONES = 20
