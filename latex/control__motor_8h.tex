Módulo de control de motores.

Este módulo controla los dos motores del robot usando\+:
\begin{DoxyItemize}
\item Pines digitales para dirección (MI0/\+MI1, MD0/\+MD1)
\item PWM para velocidad (PC8, PC9)
\item Timer TIM3 para generar señales PWM
